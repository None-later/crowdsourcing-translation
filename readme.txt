line 3-18: Reference information
We represent each reference in 4 lines
1: Original Ref
2: Tokenized Ref
3: TF model of Ref
4: TFIDF model of Ref

line 19 - last:
translation and Edits information
We represent each trans or edit in 5 lines
1: Turker ID
2: Original Trans/Edits
3: Tokenized Trans/Edits
4: TF model of Trans/Edits
5: TFIDF model of Trans/Edits

If Trans/Edits or Turker not exist, we use n/a represent.
